## Regels:

1. **AI Gebruiksbeleid:**
   - AI-gebruik is toegestaan maar NIET voor het schrijven van code
   - GitHub Copilot moet op "as" modus staan in plaats van "edit" modus
   - Andere LLMs (ChatGPT, etc.) mogen alleen resources, uitleg en hulp bij bug-zoeken bieden
   - GEEN code-generatie door AI-tools

2. **Vergaderingen:**
   - Alle leden moeten op tijd zijn voor de ochtend scrum meetings
   - Afwezigheid/te laat komen moet van tevoren gecommuniceerd worden via Discord server

3. **Communicatie & Updates:**
   - Alle updates moeten gepost worden in het <#1470361005099057235> kanaal op Discord
   - Alle gevonden EN opgeloste bugs moeten gepost worden in <#1470361263635697782> kanaal
   - Communicatie moet primair via Discord gebeuren, tenzij persoonlijk
   - Alle code-updates moeten naar de GitHub repository gepushed worden

4. **Code Kwaliteit:**
   - Commentaar moet toegevoegd worden om complexe code secties te verduidelijken
   - Commentaar moet anderen helpen functionaliteit te begrijpen tijdens debuggen
   - Duidelijke documentatie van code-aanpassingen

5. **Conflict Oplossing:**
   - Algemene meningsverschillen: Bespreek met de hele groep
   - Persoonlijke conflicten: Escaleer eerst naar teamleider
   - Onopgeloste kwesties: Teamleider betrekt docenten

6. **Gevolgen:**
   - Regel overtredingen worden eerst door teamleider aangepakt
   - Indien niet opgelost met teamleider, worden docenten betrokken
